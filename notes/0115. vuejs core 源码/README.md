## 2. 🫧 评价

源码无疑是最权威的学习资料，上手难度自然也是最高的。

一些建议：

- 优先刷 Vuejs 官方文档来学习 Vue 的基本使用及相关核心概念
- 如果有能力 + 有时间：可以在刷过官方文档之后，直接读源码进一步加深对 Vue 的实现原理的理解
- 如果只是在学习过程中遇到一些困惑，不清楚 Vue 的内部实现机制，想要适当深入了解部分实现原理，可以利用 DeepWiki 通过 AI 对话的方式来快速获取源码实现层面的解答，权威性会更高一些，当前 TNotes.vue 知识库中的大量涉及到 Vue 内部实现的笔记，都是借助 DeepWiki 来完成的
