## 5. 🤔 如果不具备必要的基础知识，直接学习 Vue 可以吗？

不是不行，但是并不推荐。

这其实是两种策略：

1. 先储备必要的基础知识点，再开始学习 Vue，能够让你在学习 Vue 的过程中更加得心应手，能够更快地理解 Vue 的核心概念和原理，并且在实际项目中能够更加得心应手。
2. 直接学习 Vue，边学边补充必要的基础知识点，这种方式可能会比较慢一些，因为你在学习 Vue 的过程中可能会遇到一些不理解的概念和原理，需要花费额外的时间去查阅资料和补充相关的基础知识点。Vue 官方并不会费口舌跟你介绍这些基础概念，它默认你已经会了。

推荐第一种策略，先把必要的基础知识点给储备好，再开始学习 Vue。不然在遇到问题的时候，你连基础的概念都不清楚，连向大模型提问都不知道怎么提。
