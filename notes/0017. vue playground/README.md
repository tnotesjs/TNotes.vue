## 1. 📝 概述

- https://play.vuejs.org/
  - Vue Playground，用于在线快速测试一些简单的 Vue demo。
  - 也可以用于调试 vue-router 的 demo，只需要在 Import Map 中加上 `"vue-router": "https://unpkg.com/vue-router@4.3.0/dist/vue-router.esm-browser.js",` vue-router 的引用即可。
