## 2. 🫧 评价

在前边儿的「基础」篇章中，已经介绍了 Vue 的一些必要基础知识，其中也提到了不少有关组件的相关内容，比如组件的注册、属性、事件等基础知识点。但在 Vue 官方文档中，组件相关的内容远不止这些，还有一些更深入的知识点，比如组件的依赖注入、异步组件、特殊元素 component 等等，这些内容虽然不如基础用法那么高频，但也是非常重要的。

需要知道我们开发的 Vue 应用，本质上就是一颗组件树，它是由一个个组件构成的。我们需要知道这些组件和组件之间是如何通信的，组件内部是如何工作的，以及在一些特殊场景下我们应该如何使用组件。深入组件章节就是为了帮助我们更好地理解和使用 Vue 组件而存在的。

## 3. 🤔 深入组件章节主要包括哪些内容？

Vue 官方文档中在「深入组件」章节主要包括以下内容：

- 注册
- Props
- 事件
- 组件 v-model
- 透传 Attributes
- 插槽
- 依赖注入
- 异步组件
- ……

这一章节主要会参考 Vue 官方文档的这一部分内容作为大纲来编写，并适当补充一些必要的内容。
